# CCNA用語集

CCNA学習で出てくるネットワーク用語を、あとから検索しやすい形で整理するためのサイトです。

## 使い方

- [用語集](/glossary/)を開く
- キーワードやカテゴリで絞り込む
- 覚えた用語は「覚えた」ボタンで記録する
- 「覚えた用語を非表示」で未習得の用語だけ確認する
- 学習中に出てきた用語をMarkdownで増やしていく

## 用語の増やし方

用語データは `docs/data/terms.ts` にあります。

```ts
{
  id: 'vlan',
  title: 'VLAN',
  category: 'スイッチング',
  summary: 'スイッチ上でブロードキャストドメインを論理的に分割する技術。',
  point: '異なるVLAN間の通信にはルーティングが必要です。',
  related: ['トランクポート', 'アクセスポート']
}
```
